# Синхронизация данных между Telegram и VK ботами

## Что это?

Система автоматической синхронизации фотографий работ между Telegram и VK ботами. Когда админ добавляет фото в TG бот, оно автоматически появляется в VK боте.

## Как работает?

1. **Админ добавляет фото в TG бот** → Фото автоматически загружается в VK → Отображается в обоих ботах
2. **Админ добавляет фото в VK бот** → Фото сохраняется в общую БД → Отображается в VK боте

## Быстрый старт

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте переменные окружения

```bash
# Telegram
BOT_TOKEN=your_telegram_token
ADMINS=123456789

# VK
VK_BOT_TOKEN=your_vk_token
VK_ADMINS=123456789

# База данных (Railway)
DATABASE_URL=postgresql://user:pass@host:5432/db
```

### 3. Запустите ботов

```bash
# Локально
python run_all.py

# На Railway - деплой автоматически запустит оба бота
```

## Использование

### Добавление фото в Telegram боте

1. Отправьте `/admin`
2. Нажмите "📸 Управление работами"
3. Нажмите "➕ Добавить фото"
4. Отправьте фото
5. Добавьте подпись (или пропустите)
6. ✅ Фото добавлено и синхронизировано с VK!

### Добавление фото в VK боте

1. Напишите "админ"
2. Нажмите "📸 Работы"
3. Нажмите "➕ Добавить"
4. Отправьте фото
5. Добавьте подпись (или пропустите)
6. ✅ Фото добавлено!

## Что синхронизируется?

✅ **Синхронизируется:**
- Фотографии работ (TG → VK)
- Подписи к фото
- Дата добавления

❌ **Пока не синхронизируется:**
- Фото из VK в TG (односторонняя синхронизация)
- Отзывы
- Заявки

## Проверка работы

1. Добавьте фото через TG бот
2. Откройте VK бот
3. Нажмите "📸 Наши работы"
4. Фото должно отображаться!

## Частые вопросы

**Нужно ли настраивать оба бота?**

Нет. Если настроен только TG бот - он работает нормально без синхронизации.

**Что делать, если фото не синхронизируется?**

1. Проверьте переменные окружения `VK_BOT_TOKEN` и `VK_ADMINS`
2. Проверьте логи бота на ошибки
3. Убедитесь, что `requests` установлен

**Где хранятся фото?**

- **Telegram**: фото хранятся на серверах Telegram (используется `file_id`)
- **VK**: фото хранятся на серверах VK (используется `attachment`)
- **База данных**: хранятся только ссылки на фото, не сами файлы
